
What is an operating system?

==>An OS is the software that directly manages a system’s hardware and resources, like the CPU, memory, and storage. 
==>The OS sits between applications and hardware and makes the connections between all your software and the physical resources that do the work.

Types of Operating Systems:

Operating systems can be categorized into several types based on their functionality and design.

Batch Operating Systems: 
==>Execute batches of jobs without user interaction.

Time-sharing or Multitasking Operating Systems: 
==>Allow multiple users to share system resources simultaneously.

Distributed Operating Systems:
==>Manage a group of distinct computers and make them appear to users as a single coherent system.

Network Operating Systems: 
==>Provide services to computers connected over a network, allowing for resource sharing.

Real-time Operating Systems: 
==>Ensure timely processing of inputs and outputs, suitable for critical applications like robotics and embedded systems.

Mobile Operating Systems: 
==>Designed specifically for mobile devices, these include operating systems like Android and iOS.

Examples of Popular Operating Systems

Desktop : Microsoft Windows,MacOS,Linux

Mobile: Android, ios

Network: Windowsserver, Unix

RealTime: Vxworks, eCos



Linux

Open Source: 
==>Linux is an open-source operating system, meaning its source code is freely available for anyone to view, modify, and distribute.

Unix-like: 
==>It is similar to Unix, which provides a familiar environment for users transitioning from other Unix-based systems.

Multi-User Capability:
==>Linux supports multiple users simultaneously, allowing them to access system resources like memory and applications without interference.

Multi-tasking:
==>It enables the execution of multiple processes at the same time, enhancing productivity and efficiency.

Portability:
==>Linux can run on various hardware platforms, making it adaptable for desktops, servers, mainframes, and embedded devices.

Hierarchical File System:
==>It organizes files in a structured directory hierarchy, facilitating easy navigation and management of files.

Security Features:
==>Linux incorporates robust security measures, including user authentication, authorization controls, and data encryption.

Shell Interface:
==>The shell serves as the command-line interface for users to interact with the operating system by executing commands and scripts.

Community Support:
==>A vibrant community contributes to its development and provides extensive documentation, tutorials, and forums for user support.

Wide Range of Distributions: 
==>There are numerous Linux distributions (distros), such as Ubuntu, Fedora, and CentOS, each tailored for specific use cases or user preferences.




Key Components of the Linux Operating System:

Kernel:
==>The core part of the Linux OS, responsible for managing hardware resources and system calls.
==>It operates in kernel mode, which allows it full access to all system resources.
==>Manages tasks such as memory management, process scheduling, device control, and system calls.
==>Supports various architectures and can be customized for specific needs.

System Libraries:
==>Special functions or programs that provide an interface for application programs to access kernel features.
==>Implement most functionalities of the operating system without requiring direct access to kernel code.
==>Facilitate easier application development by providing pre-written code for common tasks.

Shell:
==>The command-line interface that allows users to interact with the kernel by entering commands.
==>Acts as a bridge between the user and the kernel, interpreting user commands and executing them.
==>Can be either a command-line shell (like Bash) or a graphical user interface (GUI).

System Utilities:
==>Essential tools and programs that perform specific tasks related to system management and configuration.
==>Include utilities for file management, process control, software installation, and network configuration.
==>Help users perform administrative tasks efficiently.

Hardware Layer:
==>Comprises all physical components of the computer such as CPU, RAM, hard drives and peripheral devices.
==>The kernel interacts directly with this layer to manage resources and ensure smooth operation.

Terminal:
==>A terminal, also known as a command-line interface (CLI), shell or console is a text-based interface for interacting with a computer operating system. 
==>It provides a way for users to enter commands and receive text-based output.

Comman: 
==>A command is a specific instruction or request given to a computer operating system or shell. 
==>It tells the computer to perform a particular action or task.

                                 **************
                                 * EDITORS    *
                                 **************

There are 2 types of editors in linux

Vim editor

Nano editor

VIM EDITOR: It is used to edit the files in linux machines, It has 3 modes

Command mode

Insert mode

Save & quit mode

To open any file in vim editor : vim filename (or) vi filename

1.COMMAND MODE: this is the default mode in vim editor, It is used to perform some actions like used to copy the data, delete the data and we can make undo and redo the changes as well.

gg : used to go to 1st line of the file

G : used to go to last line of the file

M : used to go to middle of the file

4gg : used to go to 4th line of the file

17gg : used to go to 17th line of the file

:23 : used to go to 23rd line of the file

:set number : used to set numbers of the file

yy : used to copy the line

4yy : used to copy 4 lines from our cursor

p : paste the copied content

10p : paste the copied content 10 times

dd : used to delete the line

5dd : used to delete 5 lines from the cursor

u : used to undo the changes

crtl + r : used to redo the changes.

/word : used to search for a word in a file

?word : used to search for a word in a file

:%s/old/new/ : used to replace

==================================================================================================================================================
2. INSERT MODE: This mode is used to insert the data or make any modifications in a file.

To go to insert mode : i

To go back to command mode : esc

To go to the ending of the line : A

To go to starting of this line : I

To create a new line above the cursor : O

To create a new line below the cursor : o

DIFFERENCE B/W COMMAND MODE KEYS & INSERT MODE.

If we perform command mode keys we will be in command mode only.

If we perform insert mode keys we will go to insert mode.

3. SAVE & QUIT MODE: This is used to save the data and quit from vim editor

To save the data - :w

To quit from vim editor - :q

To quit forcefully - :q!

To save & quit at a time - :wq

To save & quit forcefully at a time - :wq!
